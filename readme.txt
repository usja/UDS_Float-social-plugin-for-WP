== Description ==
Simple free Wordpress plugin that shows vertical float bar with four social sites: 
- facebook;
- vkontakte;
- instagram;
- pinterest.
It's sticky and light. Don't use images.

== Installation ==

1. Upload folder `uds_float-social` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Inserft your urls to networks in `configs` `Float Social`
